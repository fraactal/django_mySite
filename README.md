
# Django MySite 


### Django basic aplication, python 3.10+

# 

* Create virtual env
* Activate Virtual env
* Install Django
* Run the app


## Create virtual env

```cmd
pip install virtualenv
python3 -m venv virtual_environment_name
```

## Activate virtual env

```cmd
source virtual_environment_name/bin/activate
```


## Install django

```cmd
pip install django
```


## Run the app

```python
$ python manage.py runserver 8008
```
